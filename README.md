# ShabuBox — Public

The public face of [ShabuBox](https://shabubox.com) — the everything-drawer that
thinks. Drop your paperwork in the box; on-device AI reads it, files it where you
would have, and answers questions with citations. Nothing leaves your Mac.

> The application source lives in a private repository while ShabuBox is actively
> maintained. This public repository hosts release artifacts, the auto-update
> appcast, issue tracking, and security disclosures. If maintenance ever truly
> lapses, the **Heirloom Guarantee** requires the source to be released under
> MPL-2.0 — see below.

## Requirements

macOS 26 or later on an Apple Silicon Mac. ShabuBox runs its AI entirely on your
Mac, which is why it needs the newer hardware — no account, no server, nothing sent
anywhere to be processed.

## The Heirloom Guarantee, briefly

ShabuBox is shipped under **The Heirloom License, Version 1.0**, variant
`HL-1.0-MPL2.0-12mo`. While the developer is active, ShabuBox is a normal paid
product with a private source repository. If there is **no maintenance signal for
12 months** — no release, no commit, no signed heartbeat — **and** a good-faith
support request goes unanswered for 90 days, the source is automatically released
under **MPL-2.0** — irrevocably, for everyone, forever. It's mechanical: no
committee, no petition, no discretion.

Even setting the license aside: your documents are ordinary files in ordinary
folders. Abandonment can never strand your data — you can open, move, and read
everything in Finder without ShabuBox at all.

Full terms: <https://shabubox.com/heirloom>.
